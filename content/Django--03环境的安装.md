### 学习Django之前的准备工作

准备好开发环境，python3.6，django2.1.8。

请确安装了python3.6或者更高的版本。并且在python下安装了virtualenv。

virtualenv安装详情请参考[Django--day01_2虚拟环境.md](.\Django--day01_2虚拟环境.md)

#### 安装Django环境

##### 1、进入虚拟环境

```python
workon my_env
```

##### 2、安装django

```python
pip install django==2.1.8
```

##### 3、退出虚拟环境

```python
deactivate
```
